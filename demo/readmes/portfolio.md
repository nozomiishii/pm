# portfolio

Personal portfolio site, built with Astro.

## Stack

- **Framework**: Astro 5
- **Styling**: Tailwind CSS
- **Content**: MDX
- **Hosting**: Cloudflare Pages

## Getting started

```sh
bun install
bun run dev
```

Open `http://localhost:4321` to view it.

## Scripts

| Command         | Description             |
| --------------- | ----------------------- |
| `bun run dev`   | Start the dev server    |
| `bun run build` | Build the site          |
| `bun run check` | Run typecheck and lint  |
